# Briefing Técnico — Site Bella Curitiba

## 1. Objetivo

Criar a primeira versão do site institucional da Bella Curitiba Panificadora, usando a identidade visual oficial da marca e conteúdo provisório validável.

A primeira versão deve servir para apresentação ao cliente e posterior ajuste com informações reais coletadas na reunião presencial.

---

## 2. Tipo de site

Site institucional one page.

A navegação do menu deve levar o usuário para seções da mesma página.

Exemplo:

- Início → #inicio
- Sobre → #sobre
- Produção artesanal → #producao
- Destaques → #destaques
- Depoimentos → #depoimentos
- Trabalhe conosco → #trabalhe-conosco
- Contato → #contato

---

## 3. Tecnologias

Usar inicialmente:

- HTML
- CSS
- JavaScript simples, se necessário

Não usar framework nesta primeira versão.

Objetivo: criar uma estrutura simples, leve, editável e fácil de subir no GitHub Pages.
